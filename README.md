## Advent of Code 2025 using C#
Welcome to my repository where I'm tackling the Advent of Code 2025 challenges using C#. This is a great way to practice and improve my skills.

## Objective
The goal of this project is to solve as many Advent of Code 2025 problems as possible using C#. This will help me gain a deeper understanding of the language and its capabilities while also sharpening my problem-solving skills.

## How to Run
To run the code, simply clone this repository and navigate to the desired day's folder. Then, compile and run the code using 'dotnet run'.
